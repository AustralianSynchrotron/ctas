\subsubsection{Detailed Description}
Class to deal with the rotation center (read \hyperlink{group__experiment_gae4e896096ce7a0b4cbfbdd88b7e56286}{DcenterOptionDesc}). This class parses the string describing variable deviation of the rotation axis from the center of the projection and then can calculate this value for any slice.
