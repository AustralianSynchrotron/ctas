\subsubsection{Detailed Description}
Projections storage. This class constructs the 3D array of the input data and then can form and output the sinograms.

If you want to build the sinograms from your own type of experiment, Build a class inheriting from the \hyperlink{classExperiment}{Experiment} class. See how it is implemented in the \hyperlink{classDEIexp}{DEIexp}, \hyperlink{classEDEIexp}{EDEIexp} constructors.
