\subsubsection{Detailed Description}
Thread distributor for the line-\/per-\/thread approach. This class collects all the information and data needed for the multithreaded backprojection and then can be used by independent threads to give them next portion of work: one line with the rotation angle of this line. I.e. it distributes the work over any number of threads.
