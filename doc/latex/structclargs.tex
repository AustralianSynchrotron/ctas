\subsubsection{Detailed Description}
Variables affected by the the command line arguments. Note that every file containing code for an executable has it's own 'clargs' structure with it's own initializer. The meaning of all these clargs is the same: variables definable by the command line options together with the parser in the form of the initializer \hyperlink{structclargs_ac5a54b5b965289a3ad0766317ae630dd}{clargs::clargs()}, while the contents varies. The Doxygen cannot correctly parse many structures/functions with the same name and produces bad output. Do {\bfseries NOT} rely on the Doxygen-\/made documentation for this structure, but read the code itself.
