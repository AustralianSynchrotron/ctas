NAME

ctas - Interface to all commands in the ctas package.  

SYNOPSIS

ctas [OPTIONS] [COMMAND [COMMAND SPECIFIC ...]]  

DESCRIPTION

ctas is the command line interface (CLI) to all commands provided by the ctas package. Any arguments or options passed after COMMAND are passed directly to the command and parsed by it.  

OPTIONS

-h, --help
Prints help message.
-?, --usage
Prints brief usage message.
-l, --list
Lists all existing commands.
